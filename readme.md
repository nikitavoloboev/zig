# Zig

> Zig libraries + testing code

## File structure

- [src](src) - testing out zig code
  - [main.zig](src/main.zig) - main entry point

## Setup

Everything is driven using [bun](https://bun.sh/) commands.

Assumes [zig](https://ziglang.org/learn/getting-started/) is installed.

## Run

```
bun run dev
```

Runs: `zig build run`
